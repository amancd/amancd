# About Me
Software Development Engineer (SDE-1) at Tap Invest

I am an engineer bridging the gap between mobile development and backend architecture. I specialize in building scalable mobile applications with Flutter and engineering robust backend microservices using Java and Spring Boot. 

My focus is on crafting seamless user experiences while optimizing backend performance and API design. Throughout my development cycles, I leverage concepts like streams, factories, and the Kotlin lifecycle to ensure high-quality software delivery. To date, I have developed and published 4 applications across the App Store and Play Store, reaching over 200,000 users.
